Medidas relativas a espaçamentos

m - margem - espaçamento entre os elementos, ou seja a parte de fora p - padding - espaçamento interno dos elementos border - borda que separa um elemento do outro

<> Os tamanhos padrões vão de 0 até 5 onde 0 é completamente sem espaçamento e 5 é o maior dos espaços
m-2 / p-2
p-0 / m-0 t - top - parte superior b - bottom - parte inferior s - start - começo do elemento e - end - final do elemento
pt-2 - padding top 2
pb-4 - padding bottom 4
ps-0 - padding start 0
pe-1 - padding end 1 Orientação por eixo
x - equivalente a start e end
y - equivalente a top e bottom
px-5 - padding start e end 5
py-5 - padding top e bottom 5

Resposnvididade por tamanhos de tela

as medidas de tamanhos de tela são aplicadas conforme os elementos são organizados as telas ex: p-xl-2 / p-sm-0
